Bienvenido a la app myWatchlist

Objetivo: Almacenar peliculas que uno desea ver para no perderlas y poder ir a buscarlas luego.
Info: Se encuentra seteado un perfil llamado "test" para probar la pagina sin necesidad de estar agregando peliculas.

Para acceder al perfil de prueba log-in en "who's there?" button con el nombre "test" y desliza hacia abajo o preciona el logo de myWatchlist para ver la lista.

"see the trailer" feature, consume una api que realiza una busqueda en youtube de la palabra que se ingresa en el titulo.
